# Решение задачи

Програма начинается с запроса числа строк в массиве. Пользователь вводит это число, далее она создает массив строк заданого размера. Далее пользователю предлагается ввести строоки для массива, и массив заполняется введенными значениями.

Далее вызывается функция FilterArrayByLength, которая принимает  введенный пользователем массив строк. Функция проходится по каждой строке в массиве,  и создает новый массив, куда будут добавлены только строки длиной не больше 3 символов.

Затем программа выводит новый отфильтрованный массив строк, у которых   длина не больше 3 символов. Каждая строка выводится на экран в цикле foreach.
